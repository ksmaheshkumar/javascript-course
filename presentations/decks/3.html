Usando tipos!

A linguagem que usamos não é tipada

Mas ainda é possível pensar em tipos como um framework para entender o código

O que são tipos?


  • Uma categorização de um valor
  • Uma afirmação lógica sobre uma expressão

Exemplos de tipos


  • Número
  • Texto
  • Booleano
  • Natural

$ \forall a. \forall b. \forall c. (b \rightarrow c) \rightarrow (a \rightarrow b) \rightarrow a \rightarrow c $
